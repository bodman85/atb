# ATB - Arbitrage Trading Bot.

I.	Before you start coding:
	1. npm run watch-js
	2. http-server dist
II.	To build .js and .css-bundle
	npm run build
III.To run application locally:
	http://localhost:8080/
IV.	To deploy changes to https://bodman85.github.io/atb/:
	1. Commit and push changes to github
	2. npm run deploy
